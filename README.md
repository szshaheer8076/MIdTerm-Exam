# Coffee Shop App

A mobile app (Expo/React Native) + backend (Node.js, Express, MongoDB) for managing a coffee shop menu.

## Features
- View all menu items
- Get a random in-stock item
- Backend hosted on Vercel
- MongoDB Atlas as database

## Tech Stack
- Frontend: Expo / React Native
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Deployment: Vercel
